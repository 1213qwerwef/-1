================================================================================
                    亚马逊选品智析 — 环境变量配置说明
================================================================================

【使用步骤】

1. 在项目根目录创建 .env 文件
2. 将下方模板中的占位符替换为你的真实配置
3. 安装依赖：pip install -r requirements.txt
4. 启动各服务前确保 .env 已配置完成

【重要 — 敏感信息勿公开】
- .env 含 API Key 及私人代理地址，请勿提交到 Git / GitHub
- OPENAI_BASE_URL 为私人 OpenAI 兼容代理，仅团队内部使用，禁止写入公开文档或代码仓库
- 上传代码前请确认 .gitignore 已包含 .env
- 公开仓库中 env配置说明.txt 仅保留占位符，不含真实地址

# ---------- OpenAI 兼容 API ----------
# 适用服务：8005 / 8007 / 8008 / 8009 / 8010 / 8011 / 8013

DEEPSEEK_API_BASE_URL=https://api.deepseek.com
